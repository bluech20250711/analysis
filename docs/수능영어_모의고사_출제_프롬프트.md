## [역할 및 전체 지시사항]

너는 20년 경력의 수능 영어영역 출제위원이다. 지금부터 실제 대학수학능력시험 영어영역과 동일한 출제 기준·형식·난이도를 따라 **모의고사 1세트(듣기 1~17번, 독해 18~45번, 총 45문항)**를 출제한다.

**전체 출제 원칙**
- 출제 기준 연도: [2027학년도 수능 대비 / 고3 6월·9월 모의평가 수준]
- EBS 연계 여부: [연계 O — 2026 수능특강 영어 지문 변형 / 연계 X — 순수 창작]
- 어휘 수준: 고3 수준(간혹 고난도 문항은 최상위권 변별용 추상적 개념어 포함 가능, 특히 21번·34번·41-42번)
- 지문은 반드시 **창작**하되, 실제 기출의 담화 전개 방식·문장 구조·논리 전개 패턴을 모델로 삼을 것 (실존 저작물·기출 지문의 표절·문장 복제 금지)
- 선택지는 항상 5지선다(단, 36·37·38번은 순서/위치 배열형 5지선다), 정답은 1개, 오답은 각각 그럴듯한 함정(부분 일치, 반대 개념, 지나친 일반화, 무관한 정보 등)으로 구성
- 문항 순서·번호·배점은 실제 수능과 동일하게 배치 (듣기 각 2점, 독해 각 2~3점, 고난도 문항 3점 관행 반영)
- 각 문항 출제 후 **정답, 근거(해당 문장/단서), 한국어 해설**을 함께 제시

## [Part 1] 듣기 1~17번 문항별 출제 기준

| 번호 | 문항 유형 | 출제 기준 |
|---|---|---|
| 1 | 목적 파악 | 한 화자(주로 안내방송/공지 형태)가 말하는 담화를 듣고 목적을 파악. 선택지는 "~하려고" 형태의 한국어 문장 5개. 담화는 3~5문장, 초반에 상황 제시 후 후반부에 목적이 드러나는 구조 |
| 2 | 의견 파악 | 남녀 대화(짧은 대화, 4~6턴) 후 한 화자의 의견을 파악. 선택지는 일반적 주장문 형태(예: "~해야 한다") |
| 3 | 관계 추론 | 대화 속 어휘·상황 단서로 두 화자의 직업적 관계 추론(예: 손님-점원, 의사-환자). 오답은 유사 직업군으로 구성 |
| 4 | 그림 불일치 | 하나의 그림(장면) 묘사 대화. 화자들이 그림 속 요소를 하나씩 언급하며 확인하는 구조. 마지막에 실제 그림과 다른 부분 1곳을 의도적으로 배치 |
| 5 | 할 일 파악 | 대화 후 남자 또는 여자가 대화 직후 할 일을 묻는 문제. 마지막 대사에서 반전이 있어야 함(중간까지의 흐름과 다른 결론) |
| 6 | 금액 계산 | 쇼핑/주문 상황, 품목별 가격 제시 후 할인·쿠폰 등 조건이 추가되어 최종 금액을 계산해야 정답 도출 가능하도록 구성 |
| 7 | 이유 파악 | 제안을 거절하거나 못하는 이유를 대화 속에서 파악. 초반 제안 → 중반 다른 이유로 오해 유도 → 후반 실제 이유 제시 |
| 8 | 언급되지 않은 것 | 특정 주제(행사, 강좌, 여행지 등)에 대해 화자가 4가지 항목을 언급, 선택지 5개 중 언급되지 않은 1개가 정답 |
| 9 | 내용 일치/불일치 | 한 화자의 안내 담화(행사 소개 등)를 듣고 5개 선택지(영어 문장) 중 내용과 다른 것 고르기 |
| 10 | 표(도표) 문제 | 표 제시 → 대화에서 2~3가지 조건(가격, 기능, 날짜 등) 언급 → 조건을 만족하는 항목 고르기 |
| 11 | 짧은 대화 응답 | 대화 마지막 화자의 말에 대한 가장 적절한 응답 선택. 대화는 2~3턴으로 짧게 |
| 12 | 짧은 대화 응답 | 11번과 유형 동일, 다른 소재(일상 대화) |
| 13 | 긴 대화 응답 | 4~6턴의 비교적 긴 대화 후 마지막 응답 선택. 논리적 흐름 반전 요소 포함 |
| 14 | 긴 대화 응답 | 13번과 동일 유형, 다른 소재 |
| 15 | 상황에 적절한 말 | 한 사람의 서술(내레이션) 형태로 상황 설명 → 마지막에 "이런 상황에서 A가 B에게 할 말로 가장 적절한 것은?" 질문. 서술은 5~7문장, 상황 맥락이 분명해야 함 |
| 16-17 | 긴 담화(공통 지문 2문항) | 하나의 긴 담화(1인 발화, 강의/설명문 형식, 8~10문장)를 듣고 두 문제 풀이. 16번=주제/목적 파악, 17번=언급되지 않은 것(또는 세부내용 불일치). 담화 내에 5개 이상의 세부 항목이 나열되는 구조 필요 |

**듣기 공통 요구사항**
- 각 문항마다 **영어 대본(스크립트) 전체**를 먼저 제시
- 대본 아래에 선택지(1~5) 제시
- 정답 번호와 정답 판단 근거 문장을 스크립트에서 직접 인용
- 발음 유사 어휘 함정(예: thirteen/thirty), 숫자 계산 함정 등 실제 수능 스타일 함정 요소 최소 1개 이상 포함

## [출력 형식 지정]

전체 45문항을 다음 순서로 출력할 것:

1. 듣기 1~17번 (문항 번호 → 스크립트 → 선택지 → 정답 → 해설)
2. 독해 18~45번 (문항 번호 → 지문[+도표/장문] → 선택지 → 정답 → 해설 → 핵심어휘)
3. 마지막에 전체 정답표(1~45번) 별도 요약

한 번에 전체 45문항을 다 만들기는 분량상 어려우므로, 다음과 같이 나눠서 요청할 것:
- 1차 요청: 듣기 1~17번
- 2차 요청("계속"): 독해 18~34번
- 3차 요청("계속"): 독해 35~45번 (장문 포함)
